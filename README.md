# 📞 Telefon Rehberi Uygulaması

Python ile geliştirilmiş, konsol tabanlı basit bir telefon rehberi uygulamasıdır.

## 🚀 Özellikler

- ➕ Kişi ekleme
- ❌ Kişi silme
- 📖 Rehberi görüntüleme
- 👥 Kayıtlı kişi sayısını görüntüleme
- 📋 Menü tabanlı kullanım
- 🚪 Programdan çıkış

## 🛠️ Kullanılan Teknolojiler

- Python 3

## 📚 Bu Projede Öğrendiklerim

Bu proje sayesinde aşağıdaki Python konularını uygulamalı olarak pekiştirdim:

- Fonksiyonlar (`def`)
- Dictionary (Sözlük) veri yapısı
- `while` ve `for` döngüleri
- `if`, `elif`, `else` karar yapıları
- Kullanıcıdan veri alma (`input`)
- Menü tabanlı konsol uygulaması geliştirme

## 🔧 Gelecekte Eklenebilecek Özellikler

- Kişi güncelleme
- İsme göre arama
- Telefon numarasına göre arama
- Verileri dosyaya kaydetme (JSON/TXT)
- Gelişmiş hata kontrolleri
